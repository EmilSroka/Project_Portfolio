Umiejętności

Słowem wstępu

Każdą z umiejętności umieściełm w jednej z 3 kategori:

  • Dobra znajomość - mam uporządkowaną wiedzę w danym temacie i mogę swobodnie się nią posługiwać (sporadycznie zaglądając do dokumentacji technicznej).
  • Standardowa znajomość - uważam, że znam tę technologię na odpowiednim poziomie, aby ubiegać się o pracą na stanowisku „junior developer”.
  • Podstawowa znajomość - mam podstawową, jeszcze nie do końca uporządkowaną wiedzę w danym temacie.

Lista umiejętności:

  • Solidne podstawy

    Dobra znajomość: fundamenty HTML i CSS, semantyczny HTML, A11y, RWD, Flex Box.

    Standardowa znajomość: animacje w CSS i SVG, CSS Grid, SCSS

    Podstawowa znajomość: SEO, canvas

  • Dobra znajomość języka JavaScript

    Dobra znajomość: fundamenty JS (scope, prototype, context, events itd.), programowanie obiektowe

    Standardowa znajomość: programowanie asynchroniczne, API przeglądarek, manipulacja DOM, ES6+

    Podstawowa znajomość: programowanie funkcyjne

  • Biblioteki frameworki

    Standardowa znajomość: Node.js, Webpack4

    Podstawowa znajomość: React

  • Wiedza ogólno informatyczna

    Dobra znajomość: Git, wyrażenia regularne

    Standardowa znajomość: algorytmy i struktury danych, powłoka Bash

    Podstawowa znajomość: testy jednostkowe
